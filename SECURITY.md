# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| latest  | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, contact the author and maintainer directly:

- **Name:** Sasank Vishnubhatla
- **Email:** <sasank@vishnubhatlas.net>

Please include:

- A description of the vulnerability
- Steps to reproduce the issue
- Any potential impact

The maintainer will work with you to understand the issue and coordinate a fix before any public disclosure.
